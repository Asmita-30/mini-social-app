# Social Post App - Backend

A Node.js/Express backend for a mini social media application with authentication, posts, likes, and comments.

## Features

- User authentication (JWT-based)
- Create posts with text and images
- Like/unlike posts
- Comment on posts
- Feed with pagination
- File upload handling
- MongoDB database

## Prerequisites

- Node.js (v16 or higher)
- MongoDB (local or Atlas)
- npm or yarn
